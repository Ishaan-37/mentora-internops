# Mentora – IIT Jammu InternOps Platform

Mentora is an internship management platform built for IIT Jammu to streamline intern coordination, mentor supervision, assignment tracking, and academic progress monitoring.

## Problem Statement

Internship programs often rely on scattered communication channels, spreadsheets, and manual follow-ups. This creates difficulties in tracking assignments, deadlines, mentor feedback, and intern progress.

Mentora centralizes the entire workflow into a single platform.

## Key Features

- Admin Dashboard
- Mentor Dashboard
- Intern Dashboard
- Internship Batch Management
- Assignment Creation & Tracking
- Deadline Monitoring
- Submission Management
- Weekly Reports
- Presentation Scheduling
- Notifications System
- Role-Based Access Control (RBAC)
- JWT Authentication
